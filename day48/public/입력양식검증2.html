정규식(regular expression)

문자열의 형식을 표현하는 수식
식 기능 설명
^ 시작 문자열의 시작을 표시
$ 끝 문자열의 끝을 표시
. 문자 문자 한 개와 일치
\d 숫자 숫자 한 개와 일치
\w 문자와 숫자 한 개의 문자 또는 숫자와 일치
\s 공백문자 공백, 탭, 줄바꿈, 캐리지리턴 문자와 일치 [] 문자 종류, 범위 ex) [abc]-> a 또는 b 또는 c 중의 하나, [a-z] -> a~z중 하나 [1-9]-> 1~9중의 하나
* 0회 이상 반복 ex) a* -> "", a, aa, aaa, ...
+ 1회 이상 ex) a+ -> a, aa, aaa, ...
? 0 또는 1회 ex) a? -> "", a
{m} m회 ex) a{3} -> aaa
{m,n} m회이상 n회이하 ex) a{1,3} -> a, aa, aaa
(ab) 그룹화 ex) (ab)* -> "", ab, abab, ababab,...
10자리 숫자의 정규식 표현 -> /^\d{10}$/
