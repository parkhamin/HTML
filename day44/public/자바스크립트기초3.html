# 자바스크립트 History 넷스케이프사의 브랜던 아이크(Brendan Eich)가 개발 처음에는 이름이 라이브스크립트(LiveScript) 최신 버전은 자바스크립트 ES6(2015년) ES6 -> ECMA(European Computer Manufacturer's Association) Script 6 유럽 컴퓨터 제조사 협회 # 자바스크립트 특징 인터프리터 방식 동적으로 타이핑 구조적인 프로그래밍을 지원 객체 기반 -> java의 특징 함수형 프로그래밍도 지원 -> C언어의 특징 # 자바스크립트 용도 HTML 컨텐트를 동적으로 변경할 수 있음. 이벤트에 반응하는 동작을 구현 가능. HTML 요소들의 속성을 동적으로 변경할 수 있음.(ex- 색상)-> CSS도 변경가능 HTML 요소들을 숨기거나 표시할 수 있음. 유저가 입력한 값들을 검증하는 작업 -> 자바스크립트를 이용 게임이나 애니메이션과 같은 반응형 컨텐츠도 구현가능
