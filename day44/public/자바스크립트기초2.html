컴파일 방식의 프로그램 -> 코드(code) 인터프리터 방식의 프로그램 -> 스크립트(script) # 실행방식 java -> 소스(코드)파일을 컴파일한 뒤 JVM(자바가상머신)에서 실행 javascript -> 브라우저가 소스 스크립트를 해석(인터프리트)해서 라인단위로 순차적으로 실행 # 작성위치 java -> 별도의 소스파일(.java) javascript -> html 파일안에 삽입 # 변수선언 java -> 변수의 타입을 반드시 선언해야함. javascript -> 변수의 타입을 선언하지 않아도 사용가능(autocasting) # 비슷한점 구문(if, for, ...), 객체지향을 지원(메서드, 클래스)
